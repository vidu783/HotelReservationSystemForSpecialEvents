# 🏨 Hotel Reservation System for Special Events  

### Hotel Reservation System
Developed a web-based hotel reservation system for special events to manage room bookings and event packages such as weddings and conferences.

The system allows users to:
- Check availability  
- Make reservations  
- Receive instant confirmations  

Hotel staff can manage bookings efficiently through a centralized dashboard.

---

## Key Features

### Hotel Reservation System
- Reservation booking and availability checking  
- Admin dashboard for booking management  
- Payment and invoice handling  
- Event package browsing and customization  
- Staff communication and coordination  
- Guest check-in and booking verification  
- Responsive user interface  

---

## Technologies & Tools

### Hotel Reservation System
- **Backend:** Java, Spring Boot  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** Microsoft SQL Server (MSSQL)  
- **Architecture:** MVC, 3-Tier Architecture  
- **Methodology:** Agile / Scrum  
- **IDE:** IntelliJ IDEA  
- **Concepts:** OOP, Software Engineering, System Design  

---

## Architecture

### Hotel Reservation System
- Presentation Layer → UI (Frontend)  
- Application Layer → Business Logic (Spring Boot)  
- Data Layer → MSSQL Database
